# NFL Player Performance Dashboard – Scope (API-Sports Edition)
- Data: API-Sports NFL API (free tier, 2025 up to Week 10)
- Environment: Google Colab (free)
- Core Features:
  - Interactive player stats (passing yards, rushing attempts, TDs, efficiency)
  - Filter by player, week (1-10), metric
  - Line/bar/scatter charts (Plotly)
  - Real-time updates via ipywidgets
- Deliverable: .ipynb notebook with full interactivity
- Stretch Goals: Efficiency calcs (YPA, TD%), ML predictions
- Source: https://api-sports.io/documentation/american-football/v1 (RapidAPI key required)
- Timeline: Oct 29 – Dec 23, 2025 (8 weeks)
- Limits: 100 req/day – cache aggressively
